Pressure Cooker

PDF Multi-Resolution Capture

How to Use Pressure Cooker

What it does

Captures PDF pages at multiple zoom levels and packages them as images in a ZIP file.

Steps

  1. Open a PDF - Use Ctrl+O or drag a PDF into your browser
  2. Click the extension icon - Opens this popup with PDF info
  3. Choose pages - Select "All Pages" or "Custom Range"
  4. Select zoom levels - 100%, 150%, 200% (or custom)
  5. Click "Capture PDF" - Wait for processing
  6. Download ZIP - Contains all captured images

Zoom Levels Explained

  • 100% - Original size view
  • 150% - 50% zoomed in
  • 200% - 2x zoomed in (for details)

Region Cropping

Each zoom level captures 4 versions of each page:

  • Full page
  • Top region
  • Middle region
  • Bottom region

This lets you capture detailed sections at different scales.

Output Format

ZIP file contains:

  • page_01/ - Images for page 1
  • page_02/ - Images for page 2, etc.
  • metadata.json - Capture details
  • README.md - Image descriptions
Each level captures full page + 3 regions (top, middle, bottom)
